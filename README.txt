Para abrir o projeto você precisa usar o Android Studio. 
Open an existing Android Studio project > Selecione o diretório onde está o projeto (planmat)
Levará algum tempo para carregar o projeto. 

Para compilar, clique no botão "Make Project" (Ctrl + F9 no Windows).
Para executar, clique no botão "Run 'app'" (Shift + F10 no Windows). Perceba que é necessário ter um emulador.

Caso não haja um emulador já instalado, clique em "Create New Emulator", selecione um modelo desejado. 
Certifique-se de que a categoria selecionada é "Phone". Clique em "next". 
Recomendamos a configuração default, Marshmallow. Clique em "next", a nova tela solicita um nome, 
além de outras configurações. Para nosso projeto basta as configurações default. Clique em "finish".
Agora você tem um emulador para selecionar e executar o projeto.
